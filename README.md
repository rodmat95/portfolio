# Portafolio de Rodrigo Chavarry

Bienvenido al repositorio de mi portafolio profesional. Este proyecto representa mi carta de presentación digital y el núcleo central de mi marca personal como desarrollador web.

## Propósito del Proyecto

El objetivo principal de este sitio web es doble:
1.  **Exhibición de Habilidades**: Demostrar en la práctica mi experiencia técnica construyendo aplicaciones web modernas, rápidas y accesibles utilizando tecnologías de vanguardia como Next.js 16 y React.
2.  **Captación de Clientes**: Servir como una herramienta de conversión para atraer a emprendedores y negocios que necesitan landing pages de alto impacto, mostrando ejemplos reales de mi trabajo y facilitando el contacto directo.

## Funcionalidades Principales

Esta aplicación web no es solo un sitio estático, sino una plataforma dinámica e interactiva que incluye:

-   **Soporte Multilingüe**: Contenido totalmente adaptado para audiencias en Inglés y Español, ampliando el alcance del profesional.
-   **Navegación Fluida**: Transiciones suaves y una experiencia de usuario optimizada, similar a una aplicación nativa.
-   **Portafolio Filtrable**: Sección de proyectos dinámica donde los visitantes pueden filtrar mis trabajos por categorías (Web, E-commerce, SaaS, etc.) para encontrar exactamente lo que buscan.
-   **Modo Oscuro/Claro**: Adaptabilidad a las preferencias del usuario con un sistema de temas visuales robusto.
-   **Formulario de Contacto Funcional**: Integración directa de correo electrónico para asegurar que cada oportunidad de negocio llegue a mi bandeja de entrada al instante.
-   **Diseño 'Mobile-First'**: Una interfaz totalmente responsiva que luce y funciona espectacularmente en dispositivos móviles, tablets y escritorio.
